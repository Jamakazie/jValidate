jValidate

jValidate is a "lightweight" jQuery plugin for form validation. This plugin is most likely incredibly inefficient and if used in heavy production servers...will uhh, well it won't be good. It also doesn't generally follow most jQuery plugin standards so there's that too.

Examples

Basic Usage

The basic usage involves creating an object to watch certain input classes. These classes have on blur events wired up

Using Groups

Groups can be used when wanting an error message to appear if a specific group needs additional information filled out.

Using a Rule

When needing a custom rule(Currently, default rules exist for required, and email value) we can check our inputs validity based on a regular expression.

Using A Precondition

We can check if an input is valid only if a precondition it met by adding a precondition flag to our element's rule

Preconditions currently do not work well with on blur events, but will be effective when calling .isValid()

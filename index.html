jValidate

jValidate is a "lightweight" jQuery plugin for form validation. This plugin is most likely incredibly inefficient and if used in heavy production servers...will uhh, well it won't be good. It also doesn't generally follow most jQuery plugin standards so there's that too.

Usage

jValidate is used in the following manner, given the html...

<form id="validate">
    <span class="container">
        <input type="text" class="email" />
        <span class="message"/>
    </span>
</form>

The javascript...

var params = {
    validate: {
        email: {
            required: true,
            email: true,
            messageElement: '.message'
        }
    }
}


var jval = $('#validate').jValidate(params); // This wires up on blur events for all effected inputs

if(jval.isValid()){
    // form is valid
}

Example 3 - Using a Rule

Example 4 - Using a Precondition

Ocassionaly we do not want our validation to be put into effect if other conditions are met, to do this, we can use a precondition

It should be noted that preconditons currently do not wire up on blur/on change events for the elements that may impact them (In this case, "Toggle me" being toggled,) However, when jval.isValid() is called, they will properly be taken into account, Future releases may try and address this issue.
